# Taste Frontend Skills

Source: https://github.com/Leonxlnx/taste-skill

Pinned source commit: `c8075169cd63d1430bbf492dd4ddd478ea9fa4da`

This directory contains the vendored Taste Skill frontend bundle used by the
Frontend and Stitch frontend specialist agents. The upstream README is preserved
as `UPSTREAM_README.md`, and the copied upstream license is available as
`LICENSE`.

## Available Skills

- `design-taste-frontend` - general high-agency frontend design.
- `gpt-taste` - stricter GPT/Codex-oriented frontend design and motion rules.
- `image-to-code` - image-first website design-to-code workflow.
- `redesign-existing-projects` - audit and improve existing UI projects.
- `high-end-visual-design` - polished calm premium UI direction.
- `minimalist-ui` - restrained editorial product UI.
- `industrial-brutalist-ui` - hard mechanical/brutalist visual direction.
- `full-output-enforcement` - guard against partial placeholder output.
- `stitch-design-taste` - Taste-style Google Stitch design system guidance.
- `imagegen-frontend-web` - image-generation prompts for web comps.
- `imagegen-frontend-mobile` - image-generation prompts for mobile screens.
- `brandkit` - image-generation prompts for brand-kit boards.
